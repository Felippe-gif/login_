# Projeto de Login com Flask

Projeto simples de **sistema de login** desenvolvido para estudos, utilizando **Python (Flask)** no back-end, **HTML e CSS** no front-end e **SQLite** como banco de dados.

O sistema permite que o usuário informe email e senha, valida os dados no banco e exibe uma mensagem (modal) quando o login está incorreto.

O projeto tem como objetivo praticar a integração entre Flask, formulários HTML, banco de dados e organização de arquivos estáticos.
